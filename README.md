# Bumperbot Workspace

A ROS 2 workspace for learning basic C++ nodes, parameters, and robot description concepts.

## Overview

This repository contains:

- simple ROS 2 C++ communication examples
- a parameter callback example
- a robot description package using URDF/Xacro and mesh files

## Packages

### `bumperbot_cpp_examples`

This package contains simple ROS 2 C++ nodes:

- `simple_publisher`
- `simple_subscriber`
- `simple_parameter`

### `bumperbot_description`

This package contains the robot model files:

- `urdf/bumperbot.urdf.xacro`
- robot mesh files in `meshes/`
- an RViz config in `rviz/display.rviz`

## Requirements

Make sure you have the following installed:

- ROS 2
- `colcon`
- `xacro`
- `robot_state_publisher`
- `rviz2` (optional, for visualization)

## Build The Workspace

```bash
cd ~/bumperbot_ws
source /opt/ros/<your_ros2_distro>/setup.bash
colcon build
source install/setup.bash
```

## Run Publisher And Subscriber

Terminal 1:

```bash
source /opt/ros/<your_ros2_distro>/setup.bash
source ~/bumperbot_ws/install/setup.bash
ros2 run bumperbot_cpp_examples simple_publisher
```

Terminal 2:

```bash
source /opt/ros/<your_ros2_distro>/setup.bash
source ~/bumperbot_ws/install/setup.bash
ros2 run bumperbot_cpp_examples simple_subscriber
```

Terminal 3:

```bash
source /opt/ros/<your_ros2_distro>/setup.bash
source ~/bumperbot_ws/install/setup.bash
ros2 topic echo /chatter
```

## Run The Parameter Node

Start the node:

```bash
source /opt/ros/<your_ros2_distro>/setup.bash
source ~/bumperbot_ws/install/setup.bash
ros2 run bumperbot_cpp_examples simple_parameter
```

Set parameters from another terminal:

```bash
source /opt/ros/<your_ros2_distro>/setup.bash
source ~/bumperbot_ws/install/setup.bash
ros2 param set /SimpleParameter simple_int_param 30
ros2 param set /SimpleParameter simple_string_param "Hi ROS 2"
```

Get parameter values:

```bash
ros2 param get /SimpleParameter simple_int_param
ros2 param get /SimpleParameter simple_string_param
```

You can also start the node with an initial integer value:

```bash
source /opt/ros/<your_ros2_distro>/setup.bash
source ~/bumperbot_ws/install/setup.bash
ros2 run bumperbot_cpp_examples simple_parameter --ros-args -p simple_int_param:=30
```

## Publish The Robot Description

Run `robot_state_publisher` with the Xacro file:

```bash
source /opt/ros/<your_ros2_distro>/setup.bash
source ~/bumperbot_ws/install/setup.bash
ros2 run robot_state_publisher robot_state_publisher --ros-args -p robot_description:="$(xacro ~/bumperbot_ws/src/bumperbot_description/urdf/bumperbot.urdf.xacro)"
```

```bash
rviz2 -d ~/bumperbot_ws/src/bumperbot_description/rviz/display.rviz
```

## Useful Commands

List workspace packages:

```bash
colcon list
```

List ROS 2 nodes:

```bash
ros2 node list
```

List ROS 2 topics:

```bash
ros2 topic list
```

Inspect a node:

```bash
ros2 node info /SimplePublisher
ros2 node info /SimpleParameter
```

## Notes

- `src/` contains the source packages
- `build/`, `install/`, and `log/` are generated by `colcon`
- `robot_state_publisher` uses the `robot_description` parameter to publish the robot TF tree
